通用定义

请求地址

[GET|POST|DELETE] /api/v1/action

请求参数

参数名 必选 类型 含义 说明
access_account 是 string 鉴权账号 登陆后返回
access_token 是 string 鉴权密码 登陆后返回
page_num 否 int 页码 第几页，默认1
page_size 否 int 页容量 每页几条，默认5
lang 否 string 语言 zh_CN/en_US

请求参数说明

这里定义的参数是通用参数，分页参数仅在查询列表时使用。

两个鉴权参数建议放在请求头中，目前做了兼容，后期将会只支持头部鉴权参数

测试环境接口地址如下，可尝试访问： http://47.75.6.39:8092/ 如果返回“hello”说明语言是英文环境，如果返回“你好”说明是中文环境， 登陆路径举例：POST --> http://47.75.6.39:8092/api/v1/login

返回数据

{
  code:0,
  msg:"success",
  result:{}
}

返回字段说明

键 含义 类型 可能的值 说明
code 操作结果 int
msg 操作结果信息 string
result 结果信息集 [object | list] 返回结果 ,根据具体定义

可能的错误

code 定义
0 操作成功
10001 系统内部错误
10002 缺少参数
10003 不支持的请求方式
10004 缺少鉴权参数
10005 登陆过期
10006 密码校验过期
10007 安全码校验过期
10008 拒绝访问

说明

所有接口请求至少都会返回 code 和 msg 这两个字段，code 的定义可用在后面获得。
